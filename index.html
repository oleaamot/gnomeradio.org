Source Code

There is available source code in the GNOME Radio repository on https://gitlab.gnome.org/ole/gnome-radio.git and http://www.gnomeradio.org/src/gnome-radio-13.0.1.tar.gz is the GNOME Radio 13.0.1 release (NPO Radio 1).

git clone https://gitlab.gnome.org/ole/gnome-radio
cd gnome-radio
autoreconf
aclocal
automake --add-missing      
./configure
make
make install
gnome-radio

GNOME Radio is available in MacPorts

port install xorg-server
port install gnome-radio
gnome-radio

You can build GNOME Radio on Fedora Core 35 with the following package dependencies installed:

sudo dnf install gnome-common
sudo dnf install intltool libtool gtk-doc geoclue2-devel yelp-tools
sudo dnf install gstreamer1-plugins-bad-free-devel geocode-glib-devel
sudo dnf install libchamplain-devel libchamplain-gtk libchamplain geoclue2

You can build GNOME Radio on Ubuntu Linux 21.10 with the following package dependencies installed:

sudo apt-get install debhelper intltool dpkg-dev-el libgeoclue-2-dev
sudo apt-get install libgstreamer-plugins-bad1.0-dev libgeocode-glib-dev
sudo apt-get install gtk-doc-tools itstool libxml2-utils yelp-tools
sudo apt-get install libchamplain-0.12-dev libchamplain-gtk-0.12
sudo apt-get install libgstreamer-plugins-bad1.0-dev libgeocode-glib-dev
